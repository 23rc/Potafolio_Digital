Proyectos

Geografía de Guatemala

Este fue uno de mis primeros proyectos, basado en un sistema educativo y creativo centrado en la geografía de mi bella Guatemala. Incluye varios botones de aprendizaje, cada uno con un temario de aprendizaje diferente.

Traductor C++

Este proyecto, desarrollado en C#, funciona como un compilador, actuando como un traductor de lenguaje. Está diseñado para traducir un lenguaje creado por el usuario al lenguaje de C++, proporcionando información sobre errores semánticos, sintácticos y léxicos, así como avisos para facilitar la corrección de posibles errores.

Proyecto de Producción de Lechones

Este proyecto Angular CLI se centra en la eficiente gestión de la producción de lechones. Desarrollado para ofrecer una solución integral, la aplicación permite el seguimiento detallado de cada etapa del proceso de producción. Desde la reproducción hasta el destete, facilita la planificación, el control de la salud y la recopilación de datos clave. Con una interfaz intuitiva, notificaciones útiles y capacidad para generar informes personalizados, esta aplicación está diseñada para mejorar la eficiencia en las operaciones de producción porcina.
